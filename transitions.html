Les animations & transitions

  • Une transition est une propriété qui change lors d'une action, un évenement

    • Exemple:

    Pour qu'un objet HTML soit animé, il faut le doter de la propriété CSS animation-name.

    Pour qu'un objet HTML soit animé, il faut le doter de la propriété CSS animation-name.

    #objet{
   width:100px;
   height:100px;
   background-color:#E80;
   animation-name:monanimation;
}


    Pour définir la séquence d'animation voulu, on aura recourt à une règle @ (dite règle at)

    Cette règle s'appelle @keyframes et c'est elle qui contiendra les styles:

    <style>
   #objet{
      width:100px;
      height:100px;
      background-color:#E80;
      animation-name:monanimation;
   }
   @keyframes monanimation{
      0%{
         width:100px;
         height:100px;
         background-color:#E80;
      }
      50%{
         width:100px;
         height:100px;
         background-color:#0A0;
         border-radius:100px;
         transform:rotate(90deg);
      }
      100%{
         width:100px;
         height:100px;
         background-color:#E80;
         transform:rotate(180deg);
      }
   }
</style>

}

    Pour faire une transformation, on met :

    .tourne {
    transform: rotate(10deg);
}

    Les différentes transformations possible :

    • Rotation : 
      rotate(30deg)
    • Echelle : 
       scale(2, 0.5)
    • Translation : 
       translate(10px, -3em)
    • Inclinaison :
      skewX(10deg), skewY(2.1rad)
    • Pour faire une figure en 3D : 
      rotateX(45deg)
    • Pour faire une transformation sur une transformation, on met : 
      transition: transform 1s
    • Pour faire une transition et une ergonomie : 
      .conteneur .bouge {
    transition: 1s;
}
.conteneur:hover .bouge {
    transform: translate(60px);
}
    • Pour faire des transitions animées en CSS : 
      .box {
  background-color: aquamarine;
}

.box:hover {
  background-color: gold;
